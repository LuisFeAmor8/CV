Sobre Mi

Estudiante con conocimientos en tecnología y programación, con interés en especializarse en full-Stack dentro de la ingeniería de software. Cuento con experiencia en el desarrollo de proyectos académicos relacionados con reconocimiento facial, bases de datos y planeación de software. Me considero una persona curiosa e innovadora, siempre en busca de nuevas soluciones tecnológicas.

Tengo experiencia en backend y frontend, utilizando lenguajes de programación como: Java, Python y JavaScript. Además, poseo conocimientos en SQL y bases de datos relacionales como MySQL y PostgreSQL, aplicando relaciones entre tablas y sólidos fundamentos en el desarrollo de bases de datos. También manejo CSS para estilos y utilizo Git y GitHub como sistema de control de versiones para la gestión eficiente del código.
